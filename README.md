# Programação Orientada a Objetos em Java
Este repositório contém exercícios, exemplos e pequenos projetos desenvolvidos durante a disciplina de **Programação Orientada a Objetos (POO) e Linguagem de Programação** utilizando a linguagem **Java**.
O objetivo é praticar os principais conceitos de programação e aplicar os fundamentos da orientação a objetos no desenvolvimento de aplicações.

## Conteúdos estudados
Durante a disciplina foram abordados os seguintes temas:
- Estruturas de decisão (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do-while`)
- Matrizes
- Encapsulamento
- Polimorfismo
- Interfaces
- Desenvolvimento de interfaces gráficas
- Implementação de operações **CRUD** (Create, Read, Update, Delete)

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- IDE utilizada nas aulas: NetBeans

## Estrutura do repositório
O repositório está organizado com exemplos e exercícios referentes aos conteúdos vistos em aula.
Cada pasta contém códigos desenvolvidos durante as aulas ou atividades propostas.
